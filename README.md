## 🛠️ **Installation Instructions**

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/iamendless10/vuejs_port.git
   ```

2. **Install Dependencies**:
   ```bash
   cd vuejs_port
   npm install
   ```

3. **Run the Project**:
   ```bash
   npm run serve
   ```
